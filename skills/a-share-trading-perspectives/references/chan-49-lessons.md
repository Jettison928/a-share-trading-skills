# 49课缠论视角

Use when the user asks for 缠论, 中枢, 背驰, 买卖点, 走势结构, or multi-timeframe technical structure.

## Core Model

- 走势由分型、笔、线段、中枢逐级构成。
- 大级别定方向，小级别找精确买卖点。
- 中枢是多空平衡区，先定位中枢，再判断突破、震荡、回抽。
- 背驰是转折线索，但背驰后的反向级别不一定等于原趋势级别。
- 标准一类、二类、三类买卖点优先于主观猜底追高。

## Checklist

1. Identify timeframe: 日线, 30分钟, 5分钟, or 1分钟.
2. Mark current structure: 笔/线段状态, 中枢上下沿, 是否扩展.
3. Check divergence: MACD柱面积、黄白线、量价是否背离.
4. Classify signal: 一类买点, 二类买点, 三类买点, or no standard point.
5. Define execution: entry price, stop level, failure condition, next review level.

## Decision Heuristics

- 下跌趋势 + 中枢完成 + 背驰确认: possible 一类买点.
- 一类买点后次级别回撤不创新低 + 缩量: possible 二类买点.
- 突破中枢上沿 + 回踩不回中枢 + 量能配合: possible 三类买点.
- 级别冲突时 wait. Do not treat a 1-minute signal as a daily signal.

## Weaknesses

- 中枢划分 has subjectivity.
- 小级别背驰 can fail inside a larger trend.
- Sudden news and limit-up/limit-down mechanics can invalidate clean technical execution.
