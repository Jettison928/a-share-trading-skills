# 赵老哥龙头打板视角

Use when the user asks about 龙头战法, 打板, 连板, 封单, 二封, market leader confirmation, or next-day limit-up discipline.

## Core Model

Trade only the mainline leader when market emotion supports offense. The framework values identification, confirmation, speed, and strict exit discipline.

## Leader Checklist

1. Mainline: policy/event/theme driver exists.
2. Sector effect: multiple stocks in the same theme strengthen together.
3. Recognition: earliest limit-up, highest height, strong turnover/volume, market attention.
4. Board quality: second seal or later is stronger than first seal;封单/成交额 should be meaningful.
5. Emotion:赚钱效应 should support attack; retreat when亏钱效应 dominates.

## Execution Checklist

- 一封: observe.
- 二封: confirm; participate only if seal quality improves.
- 三封: strong confirmation but avoid overcrowded late chase.
- Position: adjust with emotion; never full-position.
- Stop: if loss reaches preset line, exit without averaging down.
- Next day: if it cannot continue strength, reduce or exit according to plan.

## Decision Heuristics

- 主流题材 + recognized leader + strong second seal: eligible.
- Non-mainline, weak seal, no sector support: pass.
- Emotion retreat or leader failed to reseal: defensive posture.
- Do not mix this with bottom-fishing logic.

## Weaknesses

- Requires real-time limit-up, order book, seal and intraday behavior data.
- A-share T+1 and overnight news risk are material.
- Seal quality can deteriorate instantly.
